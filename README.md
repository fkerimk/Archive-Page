# Archive Page
A simple web application for browsing and displaying file previews.

### Features
Image, video, and audio players; PDF and 3D model viewers; *epic* background music support.

## Usage
Place all the files to be listed in the root directory and run `python build.py`.

To add tags to your file, simply add a `.tags` file with the same name; to add a source file, add a `.tar.xz` file; and to add a details window, add a `.md` file.

For example, you can take a look at [welcome.md](./root/welcome.md).


> [!NOTE]  
> Don't forget to run `python build.py` every time you update a file, tag, source, or detail!
